# Falck Studios

Official homepage for Falck Studios — a creative technology studio.

## Development

Open `index.html` in a browser or use a local dev server:

```bash
npm run dev
```

## Structure

```
├── index.html        # Main HTML document
├── css/style.css     # Styles and design system
├── js/main.js        # JavaScript
├── images/           # Image assets
└── .nojekyll         # Skip Jekyll processing on GitHub Pages
```
