# Agent Instructions

## General Instructions

- Never assume you have the latest knowledge, especially when it comes to AI models, endpoints etc. 
    - Search for the latest info
- Avoid overcomplicating and redundancy 

## Running and Testing

- Start the application by running `npm run dev`
- Use the playwright skill to test the application and looking at the endresult

## Working with issues

- Use the `gh` cli tool for interacting with GitHub Issues
- If an issue have sub-issues, be sure to update these as well, and if you're closing the main issue, consider closing the sub-issues too.
